# rustdesk配置和使用手册
## 安装rustdesk软件
下载地址
```
https://dl.maxd.cloud:8443/rustdesk/
```
根据自己的操作系统和平台选择相应的安装包
- windows下载 .exe结尾的文件
- 安卓下载 .apk结尾的文件
- ubuntu下载 .deb结尾的文件
- 苹果mac系统下载 .dmg结尾的文件，Intel处理器的mac下载带x86_64名称的文件，M系列处理器下载带aarch64名称的文件

下载完成后，安装软件
配置服务地址
> 控制端和被控制端都需要进行配置

windows系统
打开rustdesk设置菜单，点击ID后面的三个点
